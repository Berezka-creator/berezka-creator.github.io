3 capabilities offered by WURFL

  1. UX_FULL_DESKTOP
    • State Capability, true if user is a fully-fledged Desktop web-browser with access to keyboard and mouse.
    • This capability helps us to see how many users are visited from a desktop and improve user experience, make a more relevant content and design of the webpage or an advertising.
  2. IS_WIRELESS_DEVICE
    • User uses wireless device:mobile phone or a tablet
    • This capability helps us to see how many users are visited from mobile devices and to adapt design of the mobile website, to make a faster loaded content that adapted to mobile version. Offer person to transfer to mobile App if necessary.
  3. RESOLUTION_HEIGHT
    • Screen height expressed in physical pixels
    • This capability helps to adapt image quality to the device and make an image load faster.
