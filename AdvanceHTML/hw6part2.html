Homework 6 - Using CSS Shapes

Purple drop floating left

This is a paragraph that will start on the right side of the purple drop as the purple drop has received the CSS class of drop and that class has the float:left CSS property being applied to it. The idea is that we will modify that class to include CSS shape and you will see that the these two paragraphs will then follow the shape of that purple drop, following the curve of the purple drop instead of just following a rectangular shape of the width and height of the image.

This is the start of the second paragraph and this paragraph will also follow the float of the image until the end of its height. When the height of the image is achieved, the paragraph will then continue normally starting from the left of the browser and continuing to the right and down.
